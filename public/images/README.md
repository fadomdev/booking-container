# Imágenes del Sistema

## Imagen de Fondo del Login

Para personalizar la imagen de fondo del login, coloca tu imagen en esta carpeta con el nombre:

```
warehouse-bg.jpg
```

### Recomendaciones:

- **Formato**: JPG o PNG
- **Resolución**: Mínimo 1920x1080 px (Full HD)
- **Orientación**: Horizontal (landscape)
- **Tamaño**: Optimizado (< 500 KB recomendado)
- **Temática**: Preferiblemente relacionada con logística, almacenes o transportes

### Ejemplo de imagen:

La imagen proporcionada muestra un almacén con camiones, que es perfecta para un sistema de gestión de reservas de horarios logísticos.

Si no se encuentra la imagen, el sistema mostrará un gradiente azul como fondo por defecto.
